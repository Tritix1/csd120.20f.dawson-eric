Glossary

Element
a tag used to represent a specific part of an HTML file
Attribute
used to change how an element functions
Nesting
this is when you put an element inside an element
Syntax
the order in which characters must be entered
HTML
a markup language that tells web browsers how to structure the web pages you visit
